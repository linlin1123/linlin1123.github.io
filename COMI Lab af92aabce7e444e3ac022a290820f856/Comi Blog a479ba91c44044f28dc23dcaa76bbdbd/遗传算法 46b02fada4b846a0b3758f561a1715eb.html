遗传算法

算法简介

遗传算法（Genetic Algorithm，GA）最早由美国 John holland于20世纪70年代提出[1]，该算法是根据大自然中生物体进化规律而设计提出的。该算法模拟达尔文生物进化论的自然选择和遗传学机理的生物进化过程的计算模型，是一种通过模拟自然进化过程搜索最优解的方法。该算法利用数学运算，通过计算机仿真将问题的求解过程转换成类似生物进化中的染色体基因的复制、交叉和变异等过程，产生一群更适应环境的个体，使种群进化到搜索空间中更好的区域，这样一代一代不断繁衍进化，最后收敛到一群最适应环境的个体，从而求得问题的最优解[2],[3]。在求解较为复杂的组合优化问题时,相对一些常规的优化算法,通常能够较快地获得较好的优化结果。

遗传算法已被人们广泛地应用于组合优化、机器学习、信号处理、自适应控制和人工生命等领域[4],[5]。表示方法如下:

如Michalewicz[6]所总结的，一般遗传算法有五个基本组成部分。

1.问题潜在解的遗传表示。

2. 一种创建潜在解的初始集合的方法。

3. 利用评价函数对解的适用性进行评级。

4. 改变后代遗传组成的遗传算子（交叉、变异、选择）。

5. 遗传算法使用的参数值（种群大小、应用遗传算子的概率等）。

遗传算法的主要流程是首先输入问题数据和设置算法的参数，通过解码生成初始种群称为父代，通过解码利用适应度函数计算所有个体的适应度值，根据适应度值找到最优个体，之后经过交叉和变异生成子代，计算父代和子代个体的适应度值，更新最优个体，之后利用选择算子选出新一代父代种群，如此往复循环，达到最大迭代次数后停止寻优，输出最优个体也就是最优解。图1是算法的流程图。

算法说明

1. 编码（Encoding）与解码（Decoding）

解编码是遗传算法中最基础也是最重要的一步，因为编码会决定搜索空间的大小，编码设计的合理，可以有效的减少计算量。对于不同的问题，往往有不同的编码方式，二进制编码是较为常用的一种编码方式，二进制编码的搜索能力比较强，但是其缺点是存在汉明悬崖；在设计编码方式的时候，需要根据实际情况进行编码，编码要能够完善的表达所需要的信息，同时考虑到在设计遗传运算时是否会出现非法解以及非法解的修复，同时还要考虑到解码的方式等，总之，没有普适的编码方式，如何实现编码，是一个需要经验积累的过程。

在Holland的工作中，使用二进制进行编码。众所周知，用于函数优化问题的二进制编码由于汉明悬崖的存在而具有严重的缺陷，汉明悬崖描述了具有较大汉明距离的一对编码属于表现型空间中具有最小距离的点的现象。例如，对01111111111和100000000属于表现型空间中的相邻点（最小欧几里德距离的点），但在基因型空间中具有最大汉明距离。要穿越汉明悬崖，所有的比特都必须同时被改变，发生交叉和变异来跨越它的概率可能非常小。在这个意义上，二进制编码不保留表现型空间中的点的局部性。对于许多实际应用程序，几乎不可能用二进制编码来表示它们的解。为了有效地实现GA，已经为特定问题创建了各种编码方法。编码方法可以分类如下：

  • 二进制编码
  • 浮点数编码
  • 整数/文字序列编码
  • 通用数据结构编码

对于函数优化和约束优化问题，浮点数编码比二进制编码和格雷编码的性能更好。由于浮点数编码方法的基因型空间的拓扑结构与表现型空间的拓扑结构相同，很容易从传统方法中借用一些有用的技术来创建一些有效的遗传算子。整数或文字序列编码适用于组合优化问题。由于组合优化问题的本质是在一定的约束条件下寻找最佳的序列或组合，因此文字序列编码可能是处理此类问题最合理的方式。对于更复杂的实际问题，建议使用适当的数据结构作为基因的等位基因以捕捉问题的本质。在这种情况下，基因可能是nnn﻿元或更复杂的数据结构。根据编码的结构，编码方法也可以分为以下两种类型：

  • 一维编码
  • 多维编码

在大多数实践中，采用一维编码方法。然而，许多实际问题都有多维结构的解，所以采用多维编码方法来解决这些问题。

2. 种群初始化（Initialization）

遗传算法是一种基于种群的随机搜索算法。因此，每个遗传算法都会维护一个由备选解构成的种群。应用遗传算法解决优化问题的第一步是产生一个初始种群。产生初始种群的标准方法是在可行域中产生随机值，并分配给每个染色体的每个基因。随机选择的目标是确保初始种群为整个搜索空间的均匀表示。如果初始种群没有覆盖搜索空间，则有可能使得未覆盖区域被搜索过程所忽略。

初始种群的大小会影响计算复杂性和空间探索能力。大量的个体能增加多样性，进而提高种群的空间探索能力。然而，个体越多，子代的计算量越大。当子代的计算时间增加时，可能只需要较少的迭代数便能找到可以接受的解；另一方面，如果一个种群的个体数量较小，则只能探索空间中的较小部分。虽然子代的计算时间减少了，但遗传算法需要更多的迭代数才能收敛得到一个与大种群相当的结果。

3. 适应度评价（Fitness evaluation）

适应度函数表明个体或解的优劣性，是算法演化过程的驱动力，同时也是进行自然选择的唯一依据。对于不同的问题，适应度函数的定义方式不同。根据具体问题，计算种群中每个个体的适应度值。当遗传算法进行搜索时，种群会随着适应度进行进化，从而形成新的种群。这时，在每一代中，适应度值高的个体也就是优秀的个体有更大的几率进行繁衍，而相对较差的个体被淘汰，以便由好的个体组成的后代被复制。

4. 进化操作（Genetic Operations）

4.1 选择

选择算子是进化算法中的主要算子之一，与达尔文式的适者生存的概念直接相关。 选择算子的主要目标是获得更好的解，通过进化算法中的两个步骤来完成：

新种群的选择：在每一代的结束，一个由备选解构成的新种群会被选择作为下一代种群。新种群可以仅从后代中选择，或是同时从后代与父代中选择。选择算子的目标是确保优良个体能存活到下一代。

繁殖：后代通过交叉或变异算子生成。在交叉中，优良个体应该有更多的机会去繁殖，以确保后代包含最好个体的基因。在变异中，选择机制关注“劣势”个体。目的是引入更好的性状，以增加劣势个体的生存能力。

选择压力：算子以它们的选择压力为特征，具体说来，选择压力定义为选择算子单独反复作用使得最优解占据整个种群的速度。高选择压力的算子会比低选择压力的算子能更快地降低种群的多样性，但同时会导致过早收敛到次优解。高选择压力会限制种群的探索能力。

在过去的二十年里，学者们提出、研究和比较了许多选择方法。常见的选择方法如下：

    精英选择（Elitist selection）指确保当前种群中的最优个体都能存活到下代的过程，并非所有个体都参与选择，在选择之前，选出最优个体直接复制到新种群中，不经过变异。这样避免了因为概率原因使得最优个体被忽略的情况，但是越多个体存活到下代，新种群的多样性就越小。

4.2 交叉

交叉是通过组合随机选择的两个或多个父代个体的基因生成新个体的过程。如果选择的是最具适应能力的个体，则选择压力可能会降低新种群的多样性而引起过早收敛。根据元数 （即使用的父个体数目），交叉算子可以分为3类。

  • 无性：子代由一个父个体产生。
  • 有性： 由两个父个体产生一个或两个个体。
  • 多亲重组： 由多于两个的父个体产生一个或多个个体。

    1. 当βββ﻿值小于1时，子代间的距离比父代间的距离更短。
    1. 当βββ﻿值大于1时，子代间的距离比双亲间的距离更长。

4.3 变异

变异是随机改变染色体基因值的过程。其主要目标是使种群引入新的基因物质，提高基因的多样性。变异以一定的概率对对子代每个个体进行，并产生变异子代。应用变异的过程应该注意不要破坏高适应度个体的优良基因。为此，变异通常以较低的概率进行。另一方面，变异概率也可与个体的适应度成反比，适应度越低的个体，变异越高。为了提高前期迭代的探索能力，变异概率可以初始为较大值，并随着时间逐渐减少直到最后一代。

  • 宏变异算子

    Jones提出一种宏变异算子，也称为无头鸡算子。这个算子用前述的交叉算子重组一个父代个体和随机产生的个体，创建一个子代。尽管用交叉来将一个个体和随机产生的个体重组，但这个过程不能称为交叉算子，因为不存在遗传的概念。这个算子仍被视为变异，是由于新的随机产生的遗传物质的引入。

5. 终止条件（Stopping Conditions）

在遗传算法中，遗传算子不断进行迭代操作，直到满足终止条件。最简单的终止条件是限制遗传算法执行的最大迭代数或是调用适应度函数的次数。这个限制不能太小，否则算法不会有充足的时间去探索未知空间。除了限制执行时间，另一个用于确定种群是否收敛的标准也经常使用。收敛可以不严格地定义为种群变得稳定的时候，换句话说，就是在种群中没有基因或表现特征改变时。 以下是常用的收敛准则。

  • 当连续几代内都没有提高时则终止。如监视最优个体的适应度，如果在一个给定大小的时间内没有显著更新，则算法可以终止。
  • 当种群中没有改变时终止。在连续几代内，基因的平均改变量太小，则算法可以终止。

import numpy as np
import matplotlib.pyplot as plt
from mpl_toolkits.mplot3d import Axes3D
import math

#目标函数
def aimfun(x):
    y=21.5+x[0]*math.sin(4*math.pi*x[0])+x[1]*math.sin(20*math.pi*x[1])
    return y

#获得染色体长度
def getchromolen(delta, upper,lower):
    maxlen=25
    chromo_len=[]
    for j in range(len(upper)):
        a=(upper[j]-lower[j])/delta[j]
        for i in range(maxlen):
            if (2**i < a) and (a < (2**(i+1))):
                chromo_len.append(i+1)
                break
    return chromo_len

#解码
def decode(x,chromo_len,upper,lower):
    y=[]
    index=0
    for i in range(len(chromo_len)):
        a = lower[i]+(int(x[index:(index+chromo_len[i])],2))*(upper[i]-lower[i])/(2**chromo_len[i]-1)
        y.append(a)
        index = index+chromo_len[i]
    return y

#种群初始化
def init(pop_num,chromo_len):
    population=[]
    for i in range(pop_num):
        pop=''
        for k in chromo_len:
            for j in range(k):
                pop=pop+str(np.random.randint(0,2))
        population.append(pop)
    return population

#适应度评估
def fitnessfun(population,aimfun,chromo_len,upper,lower,fun):
    value=[]
    for i in range(len(population)):
        valuea=aimfun(decode(population[i],chromo_len,upper,lower))
        value.append(valuea)
    if fun==0:
        minvalue=min(value)
        value=[(i-minvalue+0.0000001) for i in value]
    elif fun==1:
        maxvalue=max(value)
        value=[(maxvalue-i+0.0000001) for i in value]
    return value

#轮盘赌选择
def roulettewheel(population,value,pop_num):
    fitness_sum=[]
    value_sum=sum(value)
    fitness=[i/value_sum for i in value]
    for i in range(len(population)):##
        if i==0:
            fitness_sum.append(fitness[i])
        else:
            fitness_sum.append(fitness_sum[i-1]+fitness[i])
    population_new=[]
    for j in range(pop_num):###
        r=np.random.uniform(0,1)
        for i in range(len(fitness_sum)):###
            if i==0:
                if r>=0 and r<=fitness_sum[i]:
                    population_new.append(population[i])
            else:
                if r>=fitness_sum[i-1] and r<=fitness_sum[i]:
                    population_new.append(population[i])
    return population_new

#单点交叉
def crossover(population_new,pc,ncross):
    a=int(len(population_new)/2)
    parents_one=population_new[:a]
    parents_two=population_new[a:]
    np.random.shuffle(parents_one)
    np.random.shuffle(parents_two)
    offspring=[]
    for i in range(a):
        r=np.random.uniform(0,1)
        if r<=pc:
            point=np.random.randint(0,int(len(parents_one[i])/2))
            off_one=parents_one[i][:point]+parents_two[i][point:]
            off_two=parents_two[i][:point]+parents_one[i][point:]
            ncross = ncross+1
        else:
            off_one=parents_one[i]
            off_two=parents_two[i]
        offspring.append(off_one)
        offspring.append(off_two)
    return offspring

#均匀变异
def mutation1(offspring,pm,nmut):
    for i in range(len(offspring)):
        r=np.random.uniform(0,1)
        if r<=pm:
            point=np.random.randint(0,len(offspring[i]))
            if point==0:
                if offspring[i][point]=='1':
                    offspring[i]='0'+offspring[i][1:]
                else:
                    offspring[i]='1'+offspring[i][1:]
            else:
                if offspring[i][point]=='1':
                    offspring[i]=offspring[i][:(point-1)]+'0'+offspring[i][point:]
                else:
                    offspring[i]=offspring[i][:(point-1)]+'1'+offspring[i][point:]
            nmut = nmut+1
    return offspring

#循环---------------------------------------------------------------------------
for i in range(gen):

    print(i)
    #适应度函数计算
    value = fitnessfun(population,aimfun,chromo_len,upper,lower,fun)
    #轮盘赌选择
    population=roulettewheel(population,value,pop_num)
    #交叉
    offspring_c=crossover(population,pc,ncross)
    #变异
